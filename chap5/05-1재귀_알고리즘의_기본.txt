## 재귀 알고리즘의 기본

1. 재귀 알아보기
  * 재귀(recursion): 어떠한 이벤트에서 자기 자신을 포함하고 다시 자기 자신을 사용하여 정의되는 경우

  * 예시) 자연수의 정의
    - 1은 자연수입니다.
    - 어떤 자연수의 바로 다음 수도 자연수입니다.

  * 재귀는 06장에서 살펴볼 '병합 정렬'과 '퀵 정렬', 09장에서 살펴볼 '이진 검색 트리'에서 다시 다룸


2. 팩토리얼 알아보기
  * 팩토리얼 factorial 문제 -> 재귀를 사용하는 대표적인 예로 양의 정수 곱을 구함

  * 팩토리얼은 양의 정수를 순서대로 곱한다는 의미로 순차 곱셈이라고도 함.

  * 예시) 팩토리얼 n!의 정의(n은 양의 정수)
    - 0! = 1
    - n > 0이면 n! = n x (n-1)!

    ➞　10!(10의 팩토리얼)은 10 x 9!로 구할 수 있고, 다시 9!은 9 x 8!로 구할 수 있음

  * math.factorial() 함수
    - math 모듈의 factorial() 함수: 팩토리얼 값을 구하는 표준 라이브러리
    - ex) math.factorial(x)는 정수 x의 팩토리얼 값을 반환함.
    - x가 정수가 아니거나 음수라면 ValueError 예외 처리를 내보냅니다.
